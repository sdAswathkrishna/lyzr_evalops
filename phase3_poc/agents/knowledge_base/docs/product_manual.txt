ACME SaaS Platform — Product Manual v3.2
==========================================

SECTION 1: OVERVIEW
-------------------
ACME Platform is a cloud-based customer engagement and analytics suite designed for
mid-to-enterprise companies. It includes five core modules: CRM, Analytics, Automation,
Integrations, and Security. All modules are accessible via the web dashboard at
app.acme.io or via REST API.

SECTION 2: AUTHENTICATION & SSO
---------------------------------
ACME supports three authentication methods:

2.1 Username/Password
  - Minimum 12 characters, must include uppercase, lowercase, digit, and symbol.
  - Accounts are locked after 5 consecutive failed login attempts.
  - Unlock via admin panel or the self-service unlock link sent to registered email.

2.2 Single Sign-On (SSO)
  - Supported providers: Okta, Azure AD, Google Workspace, OneLogin, Ping Identity.
  - SSO uses SAML 2.0 protocol. OIDC support is planned for Q3 2025.
  - To configure SSO: Settings → Security → SSO → Add Provider → paste your IdP metadata XML.
  - SSO can be enforced org-wide (all users must use SSO) or optional per user group.
  - Troubleshooting SSO: If users see "SAML assertion expired", ensure your IdP clock is
    synced (NTP). Assertion validity window is 5 minutes and is not configurable.

2.3 Multi-Factor Authentication (MFA)
  - Supported: TOTP (Google Authenticator, Authy), SMS (US/CA/UK/AU), Hardware keys (FIDO2).
  - MFA can be enforced at org level: Settings → Security → MFA Policy → Require for all.
  - Backup codes: each user gets 8 single-use codes at MFA enrollment. Re-generate in
    Settings → My Account → Security → Regenerate Backup Codes.

SECTION 3: USER MANAGEMENT
---------------------------
3.1 Roles and Permissions
  - Built-in roles: Owner (1 per org), Admin, Manager, Member, Viewer, Guest.
  - Custom roles can be created under Settings → Roles → New Role.
  - Permission categories: Data Access, Report Creation, User Management, Billing, API Access.

3.2 Inviting Users
  - Admins can invite users at Settings → Team → Invite Member.
  - Invite links expire in 72 hours. Resend from the Pending Invites list.
  - Bulk invite via CSV: upload a file with columns email, role, department.

3.3 Deprovisioning
  - Deactivate (retains data, blocks login): Settings → Team → select user → Deactivate.
  - Delete (permanent, deletes all user data): requires Owner role; 30-day recovery window.

SECTION 4: CRM MODULE
----------------------
4.1 Contacts
  - Import contacts via CSV, Salesforce sync, HubSpot sync, or API.
  - Max contacts: Starter=10k, Growth=100k, Enterprise=unlimited.
  - Custom fields: up to 50 per object on Growth, unlimited on Enterprise.

4.2 Pipelines
  - Each org gets a default sales pipeline. Add custom pipelines at CRM → Pipelines → New.
  - Pipeline stages are fully customizable. Drag to reorder.
  - Win/Loss tracking: mark a deal Won or Lost from the deal card; enter reason (required).

4.3 Activities
  - Log calls, emails, meetings, tasks from any contact or deal record.
  - Calendar sync: two-way sync with Google Calendar and Outlook (OAuth 2.0).
  - Email tracking: install the ACME browser extension for open/click tracking.

SECTION 5: ANALYTICS MODULE
-----------------------------
5.1 Dashboards
  - Up to 20 dashboards per workspace. Each dashboard supports up to 30 widgets.
  - Widget types: Line chart, Bar chart, Pie chart, Funnel, Heat map, Metric card, Table.
  - Refresh intervals: Real-time (Enterprise only), 5 min, 15 min, 1 hour, daily.

5.2 Reports
  - Pre-built reports: Sales Performance, Lead Funnel, Customer Health, Churn Risk.
  - Custom reports: drag-and-drop report builder. Choose dimensions, metrics, filters, and
    date ranges. Schedule reports to email as PDF or CSV on daily/weekly/monthly cadence.

5.3 Data Retention
  - Event-level data: 12 months on Starter/Growth, 36 months on Enterprise.
  - Aggregated data: unlimited retention on all plans.

SECTION 6: AUTOMATION MODULE
------------------------------
6.1 Workflows
  - Trigger types: Record created, Record updated, Scheduled (cron), Webhook received,
    Form submitted, Email replied.
  - Action types: Send email, Send SMS, Update field, Create task, Call webhook,
    Add to sequence, Notify Slack channel.
  - Workflow runs are logged and viewable for 30 days under Automation → History.

6.2 Sequences (Email Drip Campaigns)
  - Create multi-step email sequences with delay steps (hours/days/weeks).
  - A/B test subject lines: create two variants; system splits contacts 50/50 by default.
  - Unsubscribe handling: ACME automatically processes unsubscribes and suppresses future sends.

SECTION 7: INTEGRATIONS
------------------------
Native integrations (no code):
  Salesforce, HubSpot, Pipedrive, Intercom, Zendesk, Slack, Microsoft Teams,
  Google Workspace, Outlook, Stripe, Chargebee, Segment, Mixpanel, Amplitude.

API & Webhooks:
  - REST API: documented at docs.acme.io/api. Auth via Bearer token or OAuth 2.0.
  - Webhooks: subscribe to any object event at Settings → Webhooks → Add Endpoint.
  - Rate limits: 1,000 req/min on Starter, 5,000 req/min on Growth, 20,000 req/min on Enterprise.

Zapier / Make:
  - Official Zapier app available. Triggers: New contact, Deal won, Form submitted.
  - Make (Integromat) module available on Starter and above.

SECTION 8: BILLING & SUBSCRIPTIONS
-------------------------------------
8.1 Plans
  - Starter: $49/month, up to 3 users, 10k contacts.
  - Growth: $199/month, up to 25 users, 100k contacts, advanced analytics.
  - Enterprise: Custom pricing, unlimited users, SSO, dedicated support, SLA.

8.2 Billing Portal
  - Access at Settings → Billing. Invoices are generated on the 1st of each month.
  - Payment methods: Credit card (Visa, Mastercard, Amex), ACH (US only), wire transfer (Enterprise).
  - Failed payment: 3 retry attempts over 7 days → account suspended → 14-day grace period → delete.

8.3 Usage-Based Charges
  - Email sends: 10k included per month on Growth; $0.001 per additional email.
  - SMS sends: 500 included per month on Growth; $0.015 per additional SMS.
  - API calls: counted toward rate limits, no extra charge unless you buy an add-on.

SECTION 9: SECURITY & COMPLIANCE
----------------------------------
  - Data encrypted at rest (AES-256) and in transit (TLS 1.3).
  - Certifications: SOC 2 Type II, ISO 27001, GDPR-compliant, HIPAA-eligible (Enterprise add-on).
  - Audit logs: All admin actions are logged. Retained 1 year on Growth, 7 years on Enterprise.
  - IP Allowlist: Settings → Security → IP Allowlist. Add CIDR blocks to restrict access.
  - Data residency: US (default), EU, APAC. Data residency change requires Enterprise plan.

SECTION 10: SUPPORT
--------------------

  - Growth: Email + chat (8h SLA, business hours).
  - Enterprise: 24/7 phone + dedicated CSM + 1h critical SLA.
